# SweetBliss

Proyecto desarrollado en Java 

SweetBliss es un sistema básico que implementa la codificación de módulos de software utilizando conexión a base de datos MySQL mediante JDBC, aplicando buenas prácticas de programación y estándares de codificación.

## Tecnologías utilizadas
- Java
- MySQL
- JDBC
- XAMPP
- NetBeans
- Git y GitHub

## Funcionalidades
- Inserción de registros
- Consulta de registros
- Actualización de registros
- Eliminación de registros

## Base de datos
El proyecto incluye un archivo `.sql` para la creación de la base de datos, el cual puede ser importado en MySQL mediante phpMyAdmin.

## Observaciones
Este proyecto fue desarrollado con fines académicos, cumpliendo los requerimientos establecidos
